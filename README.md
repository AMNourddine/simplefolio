# NourddinePortfilio

A minimal portfolio template built with Parcel and Bootstrap.

## Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/cobiwave/NourddinePortfilio
cd NourddinePortfilio
npm install
```

Start the development server with:

```bash
npm start
```

Build the project for production:

```bash
npm run build
```

Update `src/index.html` with your own content and images.
